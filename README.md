## About Messenger

<ol>
  <li>Sent messages.</li>
  <li>Sent Images.</li>
  <li>Receiving messages in real time (Currently does not work with images).</li>
  <li>Add/Remove Users to Contacts.</li>
  <li>Block/Unblock Users.</li>
  <li>Edit profile and change password.</li>
</ol>
